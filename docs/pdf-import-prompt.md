# 考研英语一 PDF 真题导入提示词

你是我的「考研英一 AI 提分训练平台」数据整理助手。请在当前项目已有结构基础上导入我提供的考研英语一 PDF，不要重构项目、不要改 UI、不要删除已有数据。

请先读取 PDF，并按年份整理以下模块：

1. 阅读训练：Text 1-4，每篇完整文章、5 道题、A/B/C/D、答案、解析、题型、错因、出题人思维、定位句精读、词汇组。
2. 完形专项：一篇语篇、20 个空、A/B/C/D、答案、题型、解析、选项辨析、错因、做题策略。
3. 翻译训练：46-50 句原文、参考译文、翻译要点、重点词组、长难句拆解、常见扣分点。
4. 写作训练：小作文题目、大作文题目、写作思路、结构、可用表达、常见扣分点、参考范文占位。

数据写入位置必须复用当前项目：

- 阅读：`src/data/papers/年份/text1.json` 到 `text4.json`
- 完形：`src/data/cloze/clozeData.json`
- 翻译：`src/data/translationItems.json`
- 写作：`src/data/writingTemplates.json`

重要规则：

- 如果 PDF 能提取真实文字，就写入真实题面和答案。
- 如果中文解析、参考译文或范文无法可靠提取，字段写“待补充”，不要编造。
- 如果 PDF 是扫描件或乱码，保留该年份模板并标注“待补充”。
- 不要把阅读文章误放进完形、翻译或写作。
- 年份 ID 按 `2023-cloze`、`2023-translation`、`2023-writing`、`2023-text1-q1` 这类格式命名。
- 更新后必须检查：真题中心能显示年份，完形/翻译/写作能按年份进入，`npm run build` 能通过。
